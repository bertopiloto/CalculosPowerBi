Exemplos de Cálculos DAX e Excel

Exemplo 1: Cálculo de Desconto

Este cálculo calcula o desconto de 50% de um produto:

Desconto_Produto = 
  VAR Desconto = Produtos[Preco] * 0.5
  RETURN Desconto

Este cálculo pode ser usado para calcular 50% do preço de um produto.

Exemplo 2: Cálculo de Total de Vendas

Este cálculo multiplica a quantidade de produtos vendidos pelo preço de cada produto:

Total_Vendas = Vendas[Quantidade] * Vendas[Preco_Produto]

Esse cálculo retorna o total de vendas baseado na quantidade vendida e no preço de cada produto.

Exemplo 3: Cálculo usando DIVIDE

A função DIVIDE evita erros de divisão por zero:

Desconto_ProdutoDivide = 
  VAR Desconto = Vendas[Preco] * 0.5
  RETURN DIVIDE(Desconto, 20, 0)

Este exemplo calcula o desconto, dividindo por 20 e retornando 0 caso o divisor seja zero.

Exemplo 4: Usando a função RELATED

Busca o preço do produto de outra tabela usando a função RELATED:

Preco_Produto = RELATED(Produtos[Preco])

Esse exemplo retorna o preço do produto da tabela Produtos para a tabela atual.

Exemplo 5: Cálculo de Margem de Lucro

Este cálculo calcula a margem de lucro de um produto ou de todas as vendas:

Lucro = Vendas[Preco] - Vendas[Custo]
Margem_Lucro = DIVIDE(Vendas[Preco] - Vendas[Custo], Vendas[Preco], 0)

A margem de lucro é calculada como a diferença entre o preço de venda e o custo, dividida pelo preço de venda.

Se você quiser calcular a margem de lucro total para todas as vendas:

Lucro_Total = SUM(Vendas[Preco]) - SUM(Vendas[Custo])
Margem_Lucro_Total = DIVIDE(SUM(Vendas[Preco]) - SUM(Vendas[Custo]), SUM(Vendas[Preco]), 0)

Este cálculo retorna a margem de lucro total, considerando o total das receitas e dos custos.

Exemplo 6: Receita Líquida com Desconto de 10%

Este cálculo aplica um desconto de 10% nas vendas:

Receita_Liq_Desconto10% = 
  SUMX(
    Vendas, 
    Vendas[Quantidade] * Vendas[Preço] * (1 - 0.10)
  )

Este cálculo calcula o total de vendas com um desconto de 10% aplicado a cada produto vendido.

Exemplo 7: Receita Líquida com Acréscimo de 50%

Este cálculo aplica um acréscimo de 50% nas vendas:

Receita_Liq_Acrecimo50% = 
  SUMX(
    Vendas, 
    Vendas[Quantidade] * Vendas[Preço] * (1 + 0.50)
  )

Este cálculo calcula o total de vendas com um acréscimo de 50% aplicado a cada produto vendido.

Exemplo 8: Cálculo de Mensalidade com Acréscimo

Este cálculo aplica um acréscimo de 10% na mensalidade do aluno:

Mensalidade_Acrecimo10% = 
  VAR MensalidadeBase = 1000
  RETURN MensalidadeBase * (1 + 0.10)

Este cálculo aplica um acréscimo de 10% sobre o valor da mensalidade base, resultando no novo valor.

Exemplo 9: Cálculo de Mensalidade com Desconto

Este cálculo aplica um desconto de 15% na mensalidade do aluno:

Mensalidade_Desconto15% = 
  VAR MensalidadeBase = 1000
  RETURN MensalidadeBase * (1 - 0.15)

Este cálculo aplica um desconto de 15% sobre o valor da mensalidade base, resultando no valor com desconto.

Exemplo 10: Cálculo de Média de Contagem

Este cálculo calcula a média dos valores presentes na coluna Contagem da tabela Planilha1:

Media_Contagem = AVERAGE(Planilha1[Contagem])

Este cálculo calcula a média dos valores na coluna "Contagem". A função AVERAGE é usada para calcular a média aritmética.

Exemplo 11: Cálculo de Porcentagem de Dívidas de Alunos

Este cálculo calcula a porcentagem das dívidas de cada aluno em relação ao total de dívidas:

  Devendo = 
  DIVIDE(
      SUM(Planilha1[Contagem]), 
      CALCULATE(SUM(Planilha1[Contagem]), ALL(Planilha1)),
      0
  )

Este cálculo usa a função DIVIDE para calcular a porcentagem das dívidas de cada aluno, dividindo a dívida de cada aluno pela soma total das dívidas, considerando todas as linhas da tabela.

Exemplo 12: Somar Valores de Entrada e Saída

Este cálculo soma os valores de entrada e saída de uma tabela:

TotalEntradaSaida = 
  SUMX(
    Planilha1,
    Planilha1[Entrada] + Planilha1[Saída]
  )

Este cálculo soma os valores das colunas "Entrada" e "Saída" na tabela e retorna o total.

Exemplo 13: Fórmula PROCV (Excel)

A fórmula PROCV (Procura Vertical) busca um valor em uma coluna e retorna um valor em uma coluna adjacente. Exemplo:

=PROCV(A2, Planilha2!A:B, 2, FALSO)

Este exemplo procura o valor na célula A2 na coluna A da Planilha2 e retorna o valor correspondente na coluna B.

Exemplo 14: Fórmula PROCX (Excel)

A fórmula PROCX é uma versão mais flexível do PROCV. Exemplo:

      =PROCX("RA", A2:A10, B2:B10, "Valor não encontrado", 0)

Este exemplo procura o valor na célula A2 na coluna A da Planilha2 e retorna o valor correspondente na coluna B. A grande vantagem do PROCX é que ele não requer a ordenação dos dados.
